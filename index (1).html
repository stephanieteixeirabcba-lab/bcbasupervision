Step 1 — Create a free project
Go to supabase.com → New project. Note your Project URL and anon key (Settings → API).
Step 2 — Run this SQL in the Supabase SQL editor:
create table if not exists user_data ( id uuid primary key default gen_random_uuid(), user_id uuid references auth.users not null, key text not null, value jsonb, unique(user_id, key) ); alter table user_data enable row level security; create policy "Users own their data" on user_data for all using (auth.uid() = user_id); create table if not exists shared_data ( key text primary key, value jsonb ); alter table shared_data enable row level security; create policy "Anyone can read shared" on shared_data for select using (true); create policy "Authenticated can write shared" on shared_data for all using (auth.role() = 'authenticated');
Step 3 — Enable Email auth
In Supabase → Authentication → Providers → Email: enable it. For testing, also disable "Confirm email" (Auth → Settings → "Enable email confirmations" off).
